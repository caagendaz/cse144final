# Artifacts Folder

This folder holds generated outputs rather than source code.

- `previous_runs/` contains the original ConvNeXt, Swin, and EVA02 experiment outputs.
- `imported_volume_metadata/` contains macOS filesystem metadata found in the imported workspace. It is preserved for completeness but is unrelated to model training.
- New Colab runs are written to `/content/drive/MyDrive/Kaggle/improved_oof_ensemble` by default.
