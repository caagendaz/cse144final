# Data Folder

The image dataset is not bundled with this project. For a local run, arrange it as:

```text
data/
|-- train/
|   |-- 0/
|   |   `-- image files
|   |-- 1/
|   |   `-- image files
|   `-- ... one folder per class
`-- test/
    `-- image files
```

The Colab defaults expect the equivalent folders at:

```text
/content/drive/MyDrive/Kaggle/train
/content/drive/MyDrive/Kaggle/test
```
